## Jalankan projek

Membutuhkan `Node.js` versi 16 keatas dan `pnpm` versi 7.

- Install dependency

```bash
pnpm i
```

- Jalankan development

```bash
pnpm run dev
```

- Build untuk production

```bash
pnpm run build
```
